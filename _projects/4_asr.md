## Overview

**Outstanding Graduation Project** (Feb 2024 -- May 2024) at South China University of Technology.

Built a speech recognition web application based on **Whisper/FunASR**, integrated **Zhipu AI** for text correction and translation.

## Key Features

- **95% recognition rate** for minority languages
- Integrated **Zhipu AI** for intelligent text correction and translation
- Extended to robot system for voice command recognition
- **LLM-prompt-based automatic behavior tree generation** for robot task execution

## Tech Stack

- Whisper / FunASR for speech recognition
- Zhipu AI API for text post-processing
- Web application frontend for user interaction
- ROS2 integration for robot command execution
